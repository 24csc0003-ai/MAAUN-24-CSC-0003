# Health Clinic Queue Manager

## 📌 What This App Does
This is a simple web application built with Flask that helps manage patient queues in a health clinic.

Features:
- Register patients
- View waiting list (FIFO order)
- Serve patients
- Track total patients served

---

## 🧠 Concepts Used
- Object-Oriented Programming (Patient class)
- Queue (FIFO using Python list)
- datetime module (timestamp registration)
- Flask web framework

---

## 💻 How to Run This App (Step-by-Step)

1. Install Python (if not installed)

2. Download or clone this repository:
   git clone https://github.com/yourusername/health-clinic-queue.git

3. Open terminal and navigate into folder:
   cd health-clinic-queue

4. Install Flask:
   pip install Flask

5. Run the app:
   python app.py

6. Open browser and go to:
   http://127.0.0.1:5000/

---

## 📂 Project Structure
- app.py → main Flask app
- models.py → Patient class (OOP)
- templates → HTML pages
